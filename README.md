# Raycasting Horror Game (raylib)

Simple grid based raycasting engine written in C using raylib.



## Features

- Grid based world
- Mouse wall editor
- Player movement
- Player rotation
- Collision detection
- Debug information
- Direction vector

## Controls

| Key | Action |
|----|------|
| W | Move forward |
| S | Move backward |
| A | Rotate left |
| D | Rotate right |
| Left Click | Place wall |
| Right Click | Remove wall |
| F11 | Toggle fullscreen |

## Current Engine State

The engine currently supports:

- 2D grid map
- Player movement
- Collision detection
- Map editing
- Debug rendering

The raycasting renderer is **not implemented yet**.
